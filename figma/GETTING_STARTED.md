## 💡 使用建议

### 编写 PRD 的最佳实践

为了让 Figma MCP Server 更好地识别组件,建议在 PRD 中:

✅ **好的写法**:
```markdown
## 登录页面

### 表单区域
- 用户名输入框: 支持手机号/邮箱
- 密码输入框: 支持显示/隐藏密码
- 登录按钮: 主要操作按钮
- 注册按钮: 次要操作按钮
```

❌ **不推荐的写法**:
```markdown
## 登录页面

用户需要输入账号和密码,然后点击提交。
```

### 关键词列表

当前版本能识别的关键词:
- **按钮**: "按钮", "button"
- **输入框**: "输入框", "input", "文本框"
- **卡片**: "卡片", "card"
